AUTOWAGEN MASTER — pause / shutdown checklist

Do this before switching off so the next session picks up cleanly · TZ UTC+2

Before you shut down the PC

  1. Save files in Cursor — Ctrl+S on open tabs (or Cursor saves automatically when enabled).
  2. Git backup (best) — PowerShell in project folder: cd C:\laragon\www\autowagen-master → git add . → git commit -m "Where I stopped (plain English)" → git push. Then code is on GitHub even if the PC dies.
  3. Project diary — Append one line to repo root CLAUDE.md → section 9 Session log (top): what you finished, what’s next, date/time UTC+2 if you know it.
  4. If you shipped a fix/feature today — Same dated note at top of docs/CHANGELOG.md + mirror docs/CHANGELOG.html (project rule).
  5. Optional — Markdown snapshot — Copy these into docs/md_backups/YYYY-MM-DD/ (create dated folder): CLAUDE.md, ROADMAP.md, HOW_TO_START_NEW_CHAT.md, docs/TRAINING_SCREENSHOTS.md, docs/CHANGELOG.md. See docs/md_backups/README.md.
  6. Optional — Database snapshot — phpMyAdmin → your DB → Export → Quick → SQL → save .sql (Site ZIP backup does not include MySQL).

When you come back

  1. Open Cursor → project autowagen-master.
  2. Read CLAUDE.md from the top → especially section 10.
  3. Start a new chat and paste the grey prompt from HOW_TO_START_NEW_CHAT.md Step 2 (or say: “Read CLAUDE §10 — continue where we stopped”).

If work was only on the live server

  • Code + DB on hosting are separate — note in §9 what you uploaded or ran in phpMyAdmin so the next chat does not assume only local Laragon.
